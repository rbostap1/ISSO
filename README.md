# ISSO - Information Systems Student Organization

A modern, responsive website for the Information Systems Student Organization (ISSO). Built with Bootstrap 5 for a professional, non-profit like appearance.

## Features

- **Modern Design**: Clean, professional design with a non-profit feel
- **Responsive**: Fully responsive layout that works on all devices
- **Bootstrap 5**: Built with the latest Bootstrap framework
- **Easy to Customize**: CSS variables for easy color and style customization
- **Accessible**: Built with accessibility best practices

## Pages

- **Home** (`index.html`): Landing page with hero section, features, stats, and upcoming events
- **About** (`about.html`): Organization history, mission, vision, and values
- **Executive Board** (`executive-board.html`): Leadership team profiles
- **Membership** (`membership.html`): Membership benefits, pricing, and signup form
- **Resources** (`resources.html`): Career resources, learning materials, and events
- **Contact** (`contact.html`): Contact form and organization contact information

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML, CSS, and JavaScript

## Customization

### Colors

Edit the CSS variables in `css/style.css`:

```css
:root {
    --primary-color: #2563eb;
    --primary-dark: #1d4ed8;
    --primary-light: #3b82f6;
    /* ... more variables */
}
```

### Content

- Update organization name, contact information, and social media links in each HTML file
- Replace placeholder content with actual executive board members
- Add real event dates and details
- Update images in the `images/` directory
