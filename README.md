## ğŸ”§ Customization Overview

### Essential Changes
1. **Organization Name**: Replace `[Your Org]` throughout all pages
2. **Contact Information**: Update email addresses and location information
3. **Page Content**: Replace all `[Placeholder Text]` with your actual content
4. **Social Media Links**: Update social media URLs in the footer
5. **Images**: Add your organization's logo and team photos

### Optional Changes
- **Colors**: Edit CSS variables in `css/style.css`
- **Icons**: Change Bootstrap icons (see https://icons.getbootstrap.com)
- **Sections**: Add or remove page sections as needed
- **Forms**: Configure forms to work with your backend or form service
